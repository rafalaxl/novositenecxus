# Necxus — Web Design de Elite Multipágina

Agência de Landing Pages e Sites Institucionais Premium focados em velocidade extrema, design dark-mode premium e atração de leads qualificados sem burocracia.

## 🚀 Diferenciais de Performance
- **Velocidade Extrema:** Carregamento completo abaixo de 1 segundo.
- **Leads Qualificados:** Atendente inteligente integrado para filtragem de orçamentos e urgência.
- **Design Exclusivo:** Acabamento em Dark Mode Premium com micro-animações interativas e cursor glow.

## 🛠️ Stack Tecnológica
- **Estrutura:** Vite Multi-page (HTML5 leve, sem frameworks pesados).
- **Estilização:** Vanilla CSS estruturado.
- **Lógica e Interatividade:** Vanilla JS nativo e performático.

## 📐 Arquitetura de Arquivos
- `index.html` — Home Page descritiva e cases.
- `sobre.html` — Filosofia de velocidade anti-burocrática.
- `portfolio.html` — Grid de projetos com efeitos 3D.
- `atendente.html` — Quiz conversacional de qualificação para WhatsApp.
- `/css/` — Estilização modular (main, layout, components, effects).
- `/js/` — Lógicas leves e efeitos visuais no DOM.
- `/assets/` — Recursos visuais e marcas dos projetos.
